VCS
  * merupakan sistem yang menyimpan dan mengelola rekaman perubahan dari source code
Git
  * merupakaan salah satu software yang bisa melakukan VCS
Github
  * Website/layanan cloud yang dapat digunakan untuk mengelola Git

Istilah di Git dan Github:
 * repo merupakan folder project kita
 * commit adalah rekaman/snapshot dari suatu folder
 * hash adalah penanda unik pada sebuah comment
 * merge menggabungkan branch
 * checkout berpindah ke commit
